## Run for start development version

```
git clone https://github.com/GrafSoul/voice-clock.git
cd voice-clock
npm install
npm start
// or
yarn install
yarn start
```

## Run for build a pre-release production app.

```
npm build
// or
yarn build
```

## Get ready files for installation on Windows

```
npm release
// or
yarn release
```
